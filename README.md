# 🚌 Smart Bus Station – Bus Ticket Booking System

A modern and responsive Bus Ticket Booking System built with pure HTML, CSS, and JavaScript.  
This project simulates a real-world bus ticket booking experience with seat selection, discount calculation, QR code generation, and printable tickets.

---

## 🚀 Features

- 🎟 Passenger Details Form
- 🪑 Interactive Seat Selection (20 seats layout)
- 💰 Automatic Ticket Price Calculation
- 👶 Free Ride for age ≤ 5
- 🧒 50% Discount for age 6–8
- 👴 30% Senior Citizen Discount (age > 65)
- 👩 50% Special Discount for Female passengers
- 📱 QR Code Generation for each ticket
- 🖨 Printable Ticket Option
- 🎨 Modern Gradient UI Design

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- QRCode.js Library

---

## 📷 How It Works

1. Enter passenger details.
2. Select destination.
3. Choose your seat.
4. Click "Generate Ticket".
5. View ticket with QR code.
6. Print ticket if needed.

---

## 🎯 Purpose of Project

This project is designed to strengthen frontend development skills and demonstrate DOM manipulation, dynamic pricing logic, and UI design concepts using JavaScript.
